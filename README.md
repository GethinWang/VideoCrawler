# 爬虫爬取ACFun（A站）的视频，下载或者播放
播放器使用的是vlc，爬取ajax请求使用了phantomjs，合并m3u8的文件使用ffmpeg，只支持在windows环境使用
，直接运行vcs/src/main/java/Main下的方法即可
